# Contributing to Logic-Forge

Thank you for helping us forge the future of symbolic logic! 

## Technical Vision
Logic-Forge aims to be a high-performance educational tool for logic and search algorithms. 

## Contribution Guidelines
1.  **Algorithmic Efficiency**: Ensure new search patterns or logic rules are optimized for time complexity.
2.  **Visualization Clarity**: Maintain the clean, deductive look of the inference traces.
3.  **Modular Logic**: Keep the rule-sets decoupled from the visualization engine.

---
**Think logically. Code precisely.**
