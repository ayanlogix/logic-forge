# Logic-Forge | Symbolic AI Deductive Engine

A high-fidelity Explainable AI (XAI) tool for visualizing and auditing symbolic logic and decision tree induction.

## 🧠 The Problem
Most modern AI is a "Black Box." While Neural Networks are powerful, they are notoriously difficult to explain. For high-stakes decisions (Finance, Healthcare, Security), researchers often turn to **Symbolic AI** and **Decision Trees**, but these require sophisticated visualization to be readable and trustable.

## ⚡ The Solution
**Logic-Forge** provides a bespoke, professional interface for the "Deductive Engine."
1.  **Topological Tree Mapping**: Automatically visualizes hierarchical logic structures.
2.  **Synaptic Deduction Paths**: High-fidelity pulses travel the exact graph edges during inference to show the machine's "train of thought."
3.  **Real-Time Trace Logs**: A tactically formatted audit trail showing every rule evaluated by the engine.
4.  **Confidence Metadata**: Quantifies the purity of each leaf node in the symbolic graph.

## 🛠️ Technical Stack
- **Dashboard**: Frosted Glass / High-Contrast Laboratory Aesthetic.
- **Engine**: Recursive Canvas Rendering (60 FPS).
- **Core Logic**: Path-Finding Inference Simulation (XAI).
- **Typography**: Space Grotesk (Branding) / IBM Plex Mono (Tactical Data).
